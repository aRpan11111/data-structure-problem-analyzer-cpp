# Data Structure Problem Analyzer (C++)

## Description
A C++ based project that analyzes the performance of common data structures
and algorithms by measuring execution time for searching, sorting, and
basic data structure operations.

## Algorithms & Data Structures Covered
- Linear Search
- Binary Search
- Bubble Sort
- STL Sort
- Stack
- Queue
- Singly Linked List

## Technologies Used
- C++
- STL
- Chrono Library

## Features
- Execution time comparison
- Real-time input
- Performance analysis
- Console-based implementation

## Learning Outcome
- Practical understanding of time complexity
- Hands-on experience with STL
- Improved algorithmic thinking
